Web Platform

Flock Study

.Net Core MVC PostgreSQL PWA Tailwind

Challenge

Create a structured and accountable way for individuals to manage prayer requests and stay consistent in their spiritual commitments.

Approach

Designed a Progressive Web App where users can submit prayer requests, commit to praying weekly, and track daily progress through structured updates backed by a relational database.

Outcome

Delivered a focused Bible study and prayer accountability platform that encourages consistency, community engagement, and measurable spiritual growth.

Visit Project

Flock Study

Technical Approach

Architecture

Structured as an ASP.NET Core MVC app with clear separation of responsibilities to keep logic maintainable and testable.

Data Design

PostgreSQL models support requests, commitments, and progress history to ensure reliable tracking over time.

Delivery

Implemented as a PWA so users can access core features quickly on mobile and desktop with app-like behavior.

Implementation Flow

  1. 1 Defined domain entities and relationships for requests, users, and weekly commitments.
  2. 2 Built focused MVC views and validation pipelines to minimize friction during request submission.
  3. 3 Added progress tracking and summary views so users can measure consistency and engagement.
